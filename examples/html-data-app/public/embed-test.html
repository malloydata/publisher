Pretend this is your own web app

Drop four lines into any page and you have a Malloy-backed dashboard. The iframe auto-resizes to fit its content via postMessage. Authentication piggybacks on browser cookies, so no token plumbing for same-tenant embeds.

The whole snippet

That's it. Two snippets — copy & paste into your own HTML:

<script src="http://localhost:4000/sdk/publisher.js"></script>
<div id="dashboard"></div>
<script>
  Publisher.embed("#dashboard", {
    src: "http://localhost:4000/environments/demo/packages/html-data-app/index.html"
  });
</script>

For cross-tenant embeds (your customer's app on a different domain), the next step is signed embed tokens — the Stripe paymentIntent shape. See docs/embedded-data-apps.md.
